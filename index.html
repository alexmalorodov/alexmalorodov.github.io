Лабораторная работа №1. Создание личного репозитория-хостинга в GitHub.

Отчёт о выполнении работы

  1. Регистрация на GitHub

    В первую очередь необходимо зарегистрироваться на GitHub. для этого:

    1. Заходим на сайт GitHub

    2. Нажимаем на кнопку Sign up (зарегистрироваться)

    3. На странице регистрации Вам предложат ввести обязательные данные:

      • имя пользователя

      • адрес электронной почты

      • пароль

      Заполняем указанные поля, нажимаем на кнопку Create an account (создать аккаунт).

    4. На следующем этапе предложат выбрать тарифный план. GitHub - полностью бесплатный сервис, однако, он предоставляет и платные возможности - например, приватные репозитории.

    5. После этого Вы попадёте на небольшой опрос, который можно просто пропустить, нажав skip this test.

    6. Далее на указанную электронную почту придет письмо, для завершения регистрации потребкется перейти по указанной ссылке

    7. Регистрация окончена. Можно приступать к работе. Моя страница.

  2. Создание собственного репозитория

    1. Для создания репозитория, заходим на сайт GitHub и в блоке "Your repositories" нажимаем кнопку "New repositories"

    2. Заполняем параметры нового репозитория. Важно, чтобы название репозитория было в виде "username.github.io", где username - имя аккаунта на GitHub. В нашем случае это будет: "alexmalorodov.github.io". Обязательно ставим галочку "Initialize this repository with a README". Затем нажимаем кнопку "Create repository".

    3. Репозиторий создан: мой репозиторий.

  3. Загрузка файлов

    1. После создания репозитория, нужно загрузить на него файлы. Для этого не Обязательно клонировать репозиторий к себе на компьютер или постигать другие ньюансы работы с Git - можно воспользоваться интерфейсом GitHub.

      Нажимаем кнопку "Upload files".

    2. Перетаскиваем файлы в появившееся поле для загрузки и нажимаем кнопку "Commit changes". Можно добавить какой-нибудь комментарий.

    3. Готово! Файлы загружены в репозитарий.

  4. Возможные проблемы

    После создания собственного репозитория, у Вас могут возникнуть некоторые проблемы. Например, у меня, спустя минут 30 после его создания, вдруг сверху экрана появилась надпись: "Your account have been flagged". Это означает, что Ваша страница была "помечена", а, по сути, это означает блокировку. В моём случае это могло произойти либо когда я пытался править Wiki, чтобы вставить ссылку на свою страницу в нашу общую группу, либо, когда я в README написать некоторую речь. На самом деле ничего страшного не произошло. Нужно написать в их техподдержку и объяснить, что Вы ничего не делали, они обязательно ответят - мне отвечали около двух суток примерно.
